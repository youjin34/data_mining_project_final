# 통화정책 전달에 따른 주택담보대출 금리 변화가 서울 아파트 거래량에 미치는 영향

### — 소비심리와 정부 규제를 중심으로 (시계열 검증)

기준금리 → 주택담보대출금리 → 서울 아파트 거래량으로 이어지는 **2단계 전달 경로**를, 시계열 정상성·자기상관을 통제한 상태에서 단계적으로 검증한 프로젝트입니다. 단순 상관에서 멈추지 않고 **가짜 관계(spurious regression)를 걸러내고 견고하게 남는 효과만** 보고하는 데 초점을 두었습니다.

분석 기간: **2015.01 ~ 2025.12 (월별 132개월)**

-----

## 핵심 질문

> 금리는 정말 서울 아파트 거래량을 움직이는가? 움직인다면 얼마나, 그리고 소비심리·정부 규제는 어떤 역할을 하는가?

이를 두 고리로 나누어 검증합니다.

1. **정책금리 전달** — 한국은행 기준금리 ↑ → 은행 주택담보대출 금리 ↑ ?
1. **수요 반응** — 대출금리 ↑ → 구입 부담 ↑ → 거래량 ↓ ?
1. **그 외 요인** — 소비심리(CCSI), 정부 규제(DSR·LTV)

-----

|변수                       |의미          |비고                                          |

|`mortgage_rate`          |주택담보대출금리    |**신규취급액 기준 5년 주기형(고정형) 가중평균 금리**            |

> 금리 위계: 한국은행 기준금리 → 은행 자금조달비용(COFIX/조달금리) → 가산금리 → **주담대 금리(본 분석: 신규취급액 기준)**

-----

## 분석 방법 (절차 순서)

```
상관분석 → ADF 정상성 → 공적분(차분 가능여부 결정) → 1차 차분
        → Granger 인과성 → 회귀분석(+자기상관 진단·HAC) → VAR · CCF 보강
```

공적분을 차분보다 **먼저** 두는 이유: 공적분 결과가 “차분을 해도 되는지”를 결정하기 때문(공적분 있으면 VECM, 없으면 차분).

-----

## 주요 결과

### ① 기준금리 → 주담대금리 : 장기 동행은 강하나 단기 전달은 약함

|기준              |값                |
|----------------|-----------------|
|수준 상관           |**+0.795**       |
|차분 상관           |+0.107           |
|Granger (기준→주담대)|비유의 (p 0.16~0.62)|
|공적분             |없음 → 차분 적용 타당    |

![수준 vs 차분](figures/trend_vs_diff.png)

수준에서 강하게 보이던 0.795는 상당 부분 2021~2022년 **공통 상승 추세**의 산물이며, 월 단위 전달력은 약합니다.

### ② 주담대금리 → 거래량 : 같은 달 효과는 유의, 예측력은 없음

|기준           |값                         |
|-------------|--------------------------|
|수준 상관        |**-0.564**                |
|차분 상관        |-0.197                    |
|회귀(차분, HAC)  |**p ≈ 0.001 (유의)**        |
|Granger / CCF|선행 비유의 / **lag 0(동시)만 유의**|

![교차상관함수](figures/ccf_combined.png)

“관계가 있다”와 “미래를 예측한다”는 다릅니다. 같은 달의 음(-) 효과는 유의하지만 선행 예측력은 없습니다.

### ③ 회귀분석 — 설명력은 낮아도 통계적으로 깨끗

|모형         |Adj. R²|Durbin-Watson    |주담대금리 효과          |
|-----------|-------|-----------------|------------------|
|수준 회귀      |0.31   |**0.77 (자기상관 ⚠)**|—                 |
|차분 회귀(본 모형)|0.03   |**2.4 (정상 ✔)**   |**p ≈ 0.001 (유의)**|

![잔차 자기상관](figures/residual_acf.png)

- 수준 회귀의 높은 설명력은 자기상관에서 비롯된 착시. 차분 회귀가 정직한 추정.
- **소비심리(CCSI)**: 수준 OLS에서 유의(p=0.016)했으나 HAC 보정 후 p=0.077로 유의성 소멸 → 비유의.
- **DSR·LTV 더미**: 모든 모형에서 비유의.
- **VIF**: 전부 5 미만(직교화 불필요).
- VAR 개별 시차 계수에서 일부 유의성이 보이나, 더미를 내생으로 넣은 설정의 개별 계수라 강건하지 않아 **채택하지 않음**.

## 결론

> 금리가 비싸지면 서울 아파트가 조금 덜 팔리는 경향은 진짜지만, 효과는 약하고 금리로 미래를 예측할 수는 없으며, 소비심리·정책 규제의 독립적 효과는 엄밀히 보정하면 확정하기 어렵다. 견고하게 남는 것은 **②의 같은 달 음(-) 효과** 하나다.

이 연구의 의의는 결과의 강도가 아니라, 사슬의 각 고리에서 **가짜 관계를 걸러내고 약한 결과까지 투명하게 보고**한 방법론적 엄밀성에 있다.

## 한계 및 향후 과제

- 주담대금리(I(1))와 거래량(I(0))의 적분 차수가 달라, 엄밀히는 **ARDL** 모형이 정석(차분 시 과대차분 여지).
- 거래량의 **연 단위 계절성**을 직접 통제하지 않음(ADF 자동 시차 12가 계절성 시사).
- 정책을 **단일 시점 더미**로 단순화(DSR·LTV 모두 여러 차례 변경됨).
- 주택가격·공급(입주물량)·전세가 등 주요 결정요인 미포함.
